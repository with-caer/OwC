This directory contains pre-built [Dev Container](https://containers.dev) images configured as [Workbenches](../workbench/).

## Getting Started

Simply add a new `.devcontainer.json` to your project referencing one of the [images listed below](#whats-here). Here's an example for the `workbench-rust` image:

```json
{
    "image": "ghcr.io/with-caer/workbench-rust:latest"
}
```
